# Codebook

Store variable definitions, data dictionaries, and identifier crosswalks here.

## Recommended files

- `variables.md` — Variable names, definitions, sources, and construction notes
- `sample_filters.md` — Sample selection criteria and observation counts at each step
- `identifier_map.md` — Crosswalk between identifier systems (PERMNO, GVKEY, CIK, FIPS, etc.)
