Advise on the best journal target for this paper:

$ARGUMENTS

Read `references/journal-targets.md` for detailed journal profiles.

Steps:
1. Classify the paper's true contribution type.
2. Apply hard filters (no broad finance contribution → skip JF/JFE/RFS; housing policy → prefer JHE; urban/spatial → prefer RSUE).
3. Score top candidates on topic fit, audience fit, novelty, method, breadth, and data readiness.
4. Produce:
   - Top 3 journal targets in ranked order with rationale
   - 5 concrete revisions to improve fit for the top target
   - 3 desk-reject risks for the top target
   - Mutable submission details to verify manually (fees, formatting, editors, data policy)
